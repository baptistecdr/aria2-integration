## Ôťů Issue #2: Zod/Mini Transform Compatibility

### Files Changed
- `src/popup/models/global-stat.ts` (1 line import change)

### Changes Made
**Before**:
```typescript

```

**After**:
```typescript
import * as z from "zod";  // Full zod@4.4.3 supports transform()

```

### Root Cause
`zod/mini` is a stripped-down version that doesn't support `.transform()` method in jsdom test environment, causing `TypeError: string(...).transform is not a function` errors.
